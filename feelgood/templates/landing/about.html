{% blocktrans %}Feelgood is a non-profit service to help you combat depression. By providing tools from Cognitive Behavioral Therapy, Feelgood helps you work through your thoughts and keep track of your depression.{% endblocktrans %}

{% blocktrans %}Feelgood is not made by CBT professionals, it is not a replacement for a professional therapist. If you have any sort of suicidal urges, contact a suicide hotline immediately, you can find a number to call here.{% endblocktrans %}

{% trans 'Privacy' %}

{% blocktrans %}Your connection to Feelgood is not encrypted. This means that the information you send to Feelgood can be intercepted along the way. As soon as we have enough donations to pay for an encryption certificate ($50), we will remedy this.{% endblocktrans %}

{% blocktrans %}The information that you store with Feelgood is not shared with any third party. There is no external statistics service or anything else that get access to any sort of information about you or the fact that you are using this website.{% endblocktrans %}

{% blocktrans %}There are system administrators that have access to the databases that store the information you have entered into Feelgood. This is solely for purposes of creating backups, upgrades and maintenence. These adminstrators have signed non disclosure agreements and are prohibited from sharing any information from the databases, as well as accessing the databases for any other purpose than system administration.{% endblocktrans %}
